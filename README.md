# Face Recognition System

A simple real-time face recognition project using Python, OpenCV, and the face_recognition library.

The program captures a reference face from the webcam and checks if the same face appears in live video.

## Features
- Capture a reference face using a webcam
- Real-time face recognition
- Displays "Identified" or "Unknown"
- Exit by pressing `q`

## Requirements
- Python 3
- Webcam
